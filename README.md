To use this project, follow the steps below:

1-> Use the command "git clone https://github.com/Haslyk/BookApp.git" to clone the project. <br>
2-> Navigate to the project directory using the command "cd <booksApp>". <br>
3-> Use the command "npm install" to install the necessary dependencies. <br>
4-> Edit the database settings in the "db.js" file located in the "config" folder. <br>
5-> Create "users" and "books" tables in your database with the following properties: <br>
      Books -> { title (string) ,description (string), author (string), year (int), cover (string) } <br>
    Users -> {fullName (string), username (string), password (string) } <br>
6-> Use the command "npm start" to start the API. <br>
7-> Test the API using Postman or a similar tool. <br>
